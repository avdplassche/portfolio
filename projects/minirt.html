Minirt

C Ray Tracing Graphics MLX
View Source Code on GitHub

MiniRT project is a fundamental **ray tracing engine** developed from scratch, serving as a demonstration of computer graphics and physically-based rendering principles.

How does it works ?

The method calculates the final color of each pixel by simulating the path of light. For every intersection with an object, the color is determined using a complex model that synthesizes data from the **object's material**, **light sources**, and **view direction**. The result is a highly realistic image that accurately captures shadows, reflections, and material properties.

The scene

Objects data are defined in a **.rt file** parsed before image rendering. The parser validates and translates simple text definitions into the complex mathematical objects required by the ray tracer.

C is for Camera

The viewpoint and direction of the scene, defined by origin, orientation vector, and FOV.

A is for Ambient Light

A uniform background illumination that brightens all objects equally, preventing absolute blackness.

L is for Lights

Point light sources defined by position, brightness, and color, critical for shadows and reflections.
